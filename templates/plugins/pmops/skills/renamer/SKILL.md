# Renamer

Use renamed.to plugin for AI-powered file organization.

Run:

```bash
claude plugin add renamed-to/plugin.renamed.to
```

Requires:
- renamed.to account
- CLI: `brew tap renamed-to/cli && brew install renamed`
- MCP fallback if CLI unavailable

Usage:
- Rename files based on content analysis
- Organize files intelligently
- Split PDFs by topic
