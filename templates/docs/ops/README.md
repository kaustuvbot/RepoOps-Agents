# Operations Docs

Repo-native project management layer.

| File / Folder | Purpose |
|---------------|---------|
| `PM_GUIDE.md` | How PM creates and manages work — read this first |
| `ROADMAP.md` | Initiatives and milestone direction |
| `task-index.csv` | The board — one row per task |
| `tasks/` | Task detail files |
| `epics/` | Epic grouping files |
| `decisions/` | Durable architectural/product decisions |
| `handoffs/` | Transfer notes for paused or reassigned work |
| `IMPLEMENTATION_PLAYBOOK.md` | Developer execution flow reference |

Core rule: `In Progress` means exactly one owner.
